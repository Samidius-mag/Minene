# Minene - Minecraft Сервер "Искатели Приключений"

Minecraft сервер с тематикой "Искатели Приключений", разделенный на три государства: Омега, Бета и Вега.

## Описание мира

Мир разделён на три государства:
- **Омега** - центр координат (0, 0)
- **Бета** - расположено к северу от Омеги
- **Вега** - расположено к югу от Омеги

### Характеристики государств:
- Каждое государство имеет размер 5000x5000 блоков (10000 блоков по периметру)
- Государства разделены лавой шириной 200 блоков (приватная территория)
- В центре каждого государства находится замок с НПС "Правитель"
- Территория замка + 200 блоков - приватная территория (нельзя строить/разрушать)
- Правителя нельзя убить, только взаимодействовать

### Система авторизации:
- При первом входе игрок должен авторизоваться (логин, пароль)
- После авторизации игрок попадает в лобби (100x100 блоков)
- В лобби находятся 3 портала, ведущие в разные государства

## Установка

### Требования:
- Java 17 или выше
- Linux сервер
- Минимум 4GB RAM
- Минимум 10GB свободного места на диске

### Шаги установки:

1. Скачайте PaperMC сервер (рекомендуемая версия 1.21 или выше):
```bash
# Для версии 1.21
wget https://api.papermc.io/v2/projects/paper/versions/1.21/builds/latest/downloads/paper-1.21-latest.jar

# Или для конкретной версии (например, 1.21.10)
wget https://api.papermc.io/v2/projects/paper/versions/1.21.10/builds/latest/downloads/paper-1.21.10-latest.jar
```

2. Создайте папку для сервера и скопируйте jar файл:
```bash
mkdir minecraft-server
cd minecraft-server
mv ../paper-1.21.10-latest.jar paper.jar
# или для другой версии:
# mv ../paper-1.21-latest.jar paper.jar
```

3. Соберите плагины (см. BUILD.md):
```bash
# Linux/Mac
chmod +x build-all.sh
# ВАЖНО: При первой сборке Maven загрузит зависимости - это может занять несколько минут!
# Не прерывайте процесс (Ctrl+C) во время загрузки!
./build-all.sh

# Windows
build-all.bat
```

4. Скопируйте собранные JAR файлы из `plugins/[PluginName]/target/` в папку `plugins/` сервера

5. Запустите сервер первый раз:
```bash
java -Xmx4G -Xms2G -jar paper.jar nogui
```

6. Остановите сервер (Ctrl+C) и настройте `server.properties` и `eula.txt`:
   - Примите EULA: измените `eula.txt`, установив `eula=true`
   - Настройте `server.properties` по необходимости

7. Запустите сервер снова:
```bash
java -Xmx4G -Xms2G -jar paper.jar nogui
```

8. Мир будет автоматически сгенерирован при первом запуске (плагин MineneWorld)
   - Для ручной генерации используйте: `/mineneworld generate` (требуются права администратора)

## Плагины

1. **MineneAuth** - Система авторизации игроков
2. **MineneLobby** - Лобби с порталами в государства
3. **MineneProtection** - Защита территорий (лава, замки)
4. **MineneRulers** - НПС правители государств
5. **MineneWorld** - Генерация границ государств и лавы

## Конфигурация

Все плагины имеют конфигурационные файлы в папке `plugins/[PluginName]/config.yml`
