# Удаление мира Minecraft сервера

## Внимание!

**Удаление мира приведет к потере всех данных:**
- Все постройки игроков
- Все предметы в инвентарях
- Все сундуки и их содержимое
- Все настройки мира
- **ВСЕ ДАННЫЕ БУДУТ БЕЗВОЗВРАТНО УТЕРЯНЫ!**

## Перед удалением

1. **Создайте резервную копию** (если нужно сохранить данные):
```bash
# Остановите сервер
# Создайте резервную копию
tar -czf world-backup-$(date +%Y%m%d-%H%M%S).tar.gz world/
```

2. **Остановите сервер** (если он запущен):
```bash
# В консоли сервера
stop
# Или нажмите Ctrl+C
```

## Удаление мира

### Вариант 1: Удаление через командную строку

```bash
# Перейдите в папку сервера
cd /path/to/minecraft-server

# Удалите папку мира
rm -rf world/

# Если есть дополнительные миры (Ад, Край)
rm -rf world_nether/
rm -rf world_the_end/
```

### Вариант 2: Удаление с подтверждением

```bash
# Безопасное удаление с подтверждением
rm -ri world/
# Нажмите 'y' для подтверждения каждого файла
# Или 'n' для пропуска
```

### Вариант 3: Удаление через файловый менеджер

Если у вас есть доступ к файловому менеджеру (например, через SSH с графическим интерфейсом):
1. Найдите папку `world/` в директории сервера
2. Удалите её

## После удаления

1. **Запустите сервер** - он автоматически создаст новый мир:
```bash
java -Xmx4G -Xms2G -jar paper.jar nogui
```

2. **Новый мир будет сгенерирован автоматически**

3. **Плагины создадут свои структуры:**
   - MineneWorld создаст границы государств и лаву
   - MineneLobby создаст лобби
   - MineneRulers создаст правителей

## Удаление только определенных файлов

Если нужно удалить только часть данных:

```bash
# Удалить только чанки (но сохранить настройки)
rm -rf world/region/

# Удалить только данные игроков
rm -rf world/playerdata/

# Удалить только данные сундуков
rm -rf world/data/
```

## Восстановление из резервной копии

Если вы создали резервную копию:

```bash
# Остановите сервер
# Удалите текущий мир
rm -rf world/

# Восстановите из резервной копии
tar -xzf world-backup-YYYYMMDD-HHMMSS.tar.gz

# Запустите сервер
```

## Полное удаление всех данных сервера

Если нужно удалить ВСЕ данные сервера (миры, плагины, настройки):

```bash
# ОСТОРОЖНО! Это удалит ВСЕ!
cd /path/to/minecraft-server

# Удалить все миры
rm -rf world/ world_nether/ world_the_end/

# Удалить данные плагинов (опционально)
rm -rf plugins/MineneAuth/plugins/MineneLobby/plugins/MineneProtection/plugins/MineneRulers/plugins/MineneWorld/

# Удалить логи (опционально)
rm -rf logs/

# Удалить краш-репорты (опционально)
rm -rf crash-reports/
```

## Проверка размера мира перед удалением

Перед удалением можно проверить размер мира:

```bash
# Размер папки мира
du -sh world/

# Детальная информация о размере
du -h world/ | sort -hr | head -20
```
